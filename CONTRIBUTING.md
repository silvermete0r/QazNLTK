# Contributing to QazNLTK
 
Thanks for your interest in contributing! This is an educational project, so all skill levels are welcome.
 
---
 
## Ways to Contribute
 
- Fix a bug
- Add a new feature or NLP method
- Improve documentation or examples
- Add tests
- Suggest ideas via Issues
---
 
## Getting Started
 
1. **Fork** the repository on GitHub
2. **Clone** your fork locally:
```bash
   git clone https://github.com/silvermete0r/qazNLTK.git
   cd qazNLTK
```
3. **Install** the package in development mode:
```bash
   pip install -e .
```
4. **Create a branch** for your change:
```bash
   git checkout -b my-feature
```
 
---
 
## Making Changes
 
- Keep it simple — no external Python dependencies
- Follow the existing code style
- Add a short docstring to any new method
- Test your changes manually before submitting
---
 
## Submitting a Pull Request
 
1. Push your branch to GitHub:
```bash
   git push origin my-feature
```
2. Open a **Pull Request** against the `master` branch
3. Briefly describe what you changed and why
